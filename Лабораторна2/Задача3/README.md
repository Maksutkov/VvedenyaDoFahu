**3. Банкомат – перевірка суми для зняття готівки. Програма перевіряє, чи**

**можна зняти введену суму з банківського рахунку:**



**− Сума ≤ 0 → помилка**

**− Сума > баланс → недостатньо коштів**

**− Інакше → операція успішна**

## **Алгоритм роботи:**



###### **Початок ⮕ Задати balance = 1000 ⮕ Введеня x ⮕ Перевiрка, чи x<0, якщо так - помилка ⮕ При не проходженнi перевiрки, перевiряэться, чи x<balance, якщо так - недостатньо коштiв ⮕ При не проходженнi перевiрки - Операцiя успiшна! Списано x, ваш баланс: balance-x ⮕ Кiнець**



## **Пояснення:**

**При запуску цього коду вам виведеться значення вашого балансу, та потрібно буде ввести числове значення для змiнноi x(скiльки ви хочете зняти грошей з рахунку), при значеннi x меньше 0 виводиться помилка, при значеннi х бiльше вашого балансу виводиться текст, що свiдчить про нестачу грошей на балансi, а при значеннi х в промiжку вiд 0 до 1000(балансу), з рахунку списуэться ця сума, та виводиться текст, який описуэ цю операцiю.**



##### **x = 850 ; -250 ; 1200**



##### **Баланс: 150 ; 1000 ; 1000**

##### **Списано: 850 ; Помилка ; Недостатньо коштiв**
