## **Завдання:**



**Калькулятор знижок у магазині. Програма визначає знижку на товар**

**залежно від суми покупки:**

**− <100 грн → 0%**

**− 100–499 грн → 5%**

**− 500–999 грн → 10%**

**− ≥1000 грн → 15%**

## **Алгоритм роботи:**



###### **Початок ⮕ Введеня x ⮕ Перевiрка, чи x<100, якщо так - знижки нема ⮕ При не проходженнi перевiрки, перевiряэться, чи x<500, якщо так - знижка 5% ⮕ При не проходженнi перевiрки, перевiряэться, чи x<1000, якщо так - знижка 10% ⮕ При не проходженнi перевiрки - знижка 15% ⮕ Кiнець**



## **Пояснення:**

**При запуску цього коду вам потрібно буде ввести числове значення для змiнноi x(сума вашоi покупки), при значеннi x вiд 0 до 99 - знижка 5%, вiд 100 до 499 - знижка 5%, вiд 500 до 999 - знижка 10%, вiд 1000 - знижка 5%.**
