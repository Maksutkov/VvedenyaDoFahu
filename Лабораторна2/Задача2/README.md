## **Завдання:**



**Перевірка віку для доступу до сайту/послуги. Розробити програму, яка**

**перевіряє, чи дозволено користувачу доступ на сайт або до послуги за віком:**



**− Вік < 13 → доступ заборонено**

**− Вік 13–17 → доступ з батьківським контролем**

**− Вік ≥ 18 → повний доступ**

## **Алгоритм роботи:**



###### **Початок ⮕ Введеня x ⮕ Перевiрка, чи x<13, якщо так оцiнка - Доступ заборонено ⮕ При не проходженнi перевiрки, перевiряэться, чи x<=17, якщо так - Доступ з батьківським контролем ⮕ При не проходженнi перевiрки - «Повний доступ» ⮕ Кiнець**



## **Пояснення:**

**При запуску цього коду вам потрібно буде ввести числове значення для змiнноi x, при значеннi x меньше 13 - виводиться Доступ заборонено, вiд 14 до 17 - Доступ з батьківським контролем, вiд 18 та бiльше - Повний доступ.**



##### **x = 17 ; 25 ; 12**
